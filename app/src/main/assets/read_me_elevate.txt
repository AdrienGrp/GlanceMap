Elevate themes read me and licenses
====================================

This source fallback file is included because Elevate.xml references
read_me_elevate.txt for reuse and resource-license information.

Upstream source:
- Elevate theme family, including Elevate and Elements
- Mapsforge theme for OpenAndroMaps
- Author: Tobias Kuehn
- Contact: http://www.eartrumpet.net/contact/
- Project page: https://www.openandromaps.org/en/legend/elevate-mountain-hike-theme

The generated build assets may include the complete upstream
read_me_elevate.txt from the downloaded Elevate ZIP. This checked-in copy keeps
the license and resource-license information available in the public source
fallback assets next to Elevate.xml.

Upstream commercial-use note:

You can use it in a commercial app etc. for free if you:
- attribute the origin, for example:
  "uses Elevate map theme by Tobias Kuehn,
  http://www.openandromaps.org/en/legend/elevate-mountain-hike-theme"
- do not change anything or use parts of it, so you can only use it if you
  include the whole contents of the ZIP file.

Upstream non-commercial-use note:

For non-commercial usage you are free to reuse everything Tobias Kuehn made if
you follow these conditions:
- attribute the origin, for example:
  "symbols/patterns/code based on Elevate theme by Tobias Kuehn,
  http://www.openandromaps.org/en/legend/elevate-mountain-hike-theme"
- do not use it for commercial purposes; for example, do not include parts of
  it or directly download it with a commercial app without clarification.
- if you use some of Tobias Kuehn's work, it has to be under the same license,
  but only Creative Commons Attribution-NonCommercial-ShareAlike 4.0. Remove
  the commercial license, because it is only valid for unmodified Elevate.
- review and follow the licenses of the resources used by the theme below.
