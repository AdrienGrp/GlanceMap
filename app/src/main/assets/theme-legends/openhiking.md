# OpenHiking

OpenHiking is a topo and hiking-oriented Mapsforge theme from openhiking.eu.

Main map cues:

- Hiking routes are enabled by default.
- Optional overlays include pilgrimage routes, unmarked hiking routes, SAC
  difficulty highlighting, seasonal closure highlighting, winter hiking,
  downhill ski tracks, cross-country ski routes, and skitour routes.
- Map detail overlays include contour lines, buildings, fences, power lines,
  protected areas, military areas, access restrictions, cycleway symbols,
  region boundaries, and OSM editorial symbols.
- Natural land cover, water, tracks, paths, roads, barriers, public transport,
  and POIs are styled for outdoor map reading.

Style notes:

- Start with hiking routes and contour lines, then enable extra route or winter
  overlays as needed.
- SAC and closure highlighting depend on the map data containing those tags.
- Hill shading is not currently advertised by this bundled theme.
