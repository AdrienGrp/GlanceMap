# Elevate / Elevate Winter

Elevate is the OpenAndroMaps outdoor theme family. It is built for hiking,
cycling, mountain biking, sightseeing, and general topo map use.

Main map cues:

- Hiking routes, cycling routes, MTB routes, waymarks, path difficulty, and
  contour lines are controlled by the theme style and overlay toggles.
- Hiking routes are emphasized in the hiking style, cycling routes in cycling
  styles, and MTB routes in the mountain bike style.
- Blue markings are commonly used for hiking route emphasis, purple/violet for
  cycling route emphasis, and magenta for MTB route emphasis.
- Paths and tracks are emphasized for outdoor use. Extra surface and access
  details can be enabled from the overlay list.
- Path difficulty, trail visibility, ladders, ropes, rungs, and via ferrata
  details appear only when the source map contains those tags.
- POIs cover landscape features, places, accommodation, food, transport,
  services, shops, emergency, and recreation where the map data contains them.
- Elevate Winter uses the same family of symbols with winter-oriented styling
  for ski and snow activity use.

Style notes:

- Pick the style closest to the activity first, then use overlays to reduce
  clutter.
- Route colors and labels depend on OpenAndroMaps data. Missing or outdated
  tags can make routes or symbols appear incomplete.
- Hill shading support depends on the selected Elevate variant and the map
  renderer setting.
