# Hike, Ride & Sight

Hike, Ride & Sight is an outdoor OpenAndroMaps theme by Jerome Seydoux, based
on Bernard Mai's Outdoor theme with influence from Elevate.

Main map cues:

- Hiking routes are blue.
- Cycling routes are violet.
- MTB routes are magenta.
- Paths, footways, mountaineering paths, tracks, road surfacing, trail surface,
  access rights, and MTB grades are controlled through overlays.
- Topography overlays include cliffs, ridges, gorges, canyons, valleys, major
  contour lines, and minor contour lines.
- POI overlays cover huts, shelters, campsites, hotels, restaurants, shops,
  health, education, public transport, urban equipment, sports, barriers,
  parkings, guideposts, cairns, and survey points.
- Winter sports can be enabled separately.
- Hill shading is supported by the current bundled XML.

Style notes:

- This theme intentionally uses one base style with many overlay toggles.
- If the map feels crowded, start by disabling POI groups or route groups you
  do not need for the current activity.
- Route colors, difficulty marks, access marks, and POIs depend on
  OpenAndroMaps tags being present in the map file.
