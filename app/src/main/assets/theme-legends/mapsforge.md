# Mapsforge

Mapsforge themes are the built-in renderer styles from the Mapsforge library.

Available styles in GlanceMap:

- Classic: default general-purpose rendering.
- OSMARender: an OpenStreetMap-like style.
- Motorider: road-oriented rendering.
- Biker: cycling-oriented rendering.
- Dark: dark map colors.
- Indigo: blue-indigo map colors.

Main map cues:

- Built-in styles are simpler than the bundled outdoor themes.
- They are useful as fallback styles or for quick contrast changes.
- They do not expose the same detailed overlay controls as Elevate, Tiramisu,
  OpenHiking, or Hike, Ride & Sight.

Style notes:

- Use these when you want a lightweight map style.
- Outdoor route detail depends on both the selected built-in style and the tags
  present in the map file.
