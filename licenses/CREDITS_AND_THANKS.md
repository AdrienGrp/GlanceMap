# Credits and Thanks

Last reviewed: 2026-05-06

This page is the user-friendly credit summary for the main projects and communities used by GlanceMap.

## Main credits

1. OpenAndroMaps
   - Recommended source for downloadable maps in the companion app.
   - Website: https://www.openandromaps.org/

2. Elevate theme (Tobias Kuehn, OpenAndroMaps ecosystem)
   - Theme family used for maps rendering and style variants.
   - Theme info: https://www.openandromaps.org/en/legend/elevate-mountain-hike-theme

3. OpenHiking
   - Additional bundled/downloadable Mapsforge theme source.
   - Website: https://www.openhiking.eu/en/downloads/mapsforge-maps

4. Tiramisu (IgorMagellan)
   - Additional bundled Mapsforge theme with hiking, MTB and cycling styles.
   - Project: https://github.com/IgorMagellan/Tiramisu

5. French Kiss (Pascal Cochet / XCTrack bundle)
   - Additional bundled IGN-style Mapsforge theme sourced from the XCTrack APK snapshot provided for local testing.

6. Hike, Ride & Sight! (Jérôme Seydoux)
   - Additional bundled Mapsforge theme focused on hiking, cycling and overlay-rich map detail.
   - Theme page: http://j.seydoux.free.fr/locus/hrs.html
   - Contact: https://arkluz.com/trk?contact

7. Voluntary themes (John Campbell)
   - Additional bundled Mapsforge theme inspired by UK Ordnance Survey map conventions.
   - Website: https://voluntary.nichesite.org/
   - Credits: https://voluntary.nichesite.org/credits.html

8. OpenStreetMap contributors
   - Map data attribution:
   - `Map data (c) OpenStreetMap contributors`
   - Copyright/licensing page: https://www.openstreetmap.org/copyright

9. Refuges.info
   - Refuge and mountain POI data imports in the companion app.
   - Attribution: `© Les contributeurs de Refuges.info`
   - Website: https://www.refuges.info/

10. Overpass API / OpenStreetMap community
   - OSM POI enrichment for some imports.
   - API endpoint used: https://overpass-api.de/api/interpreter

11. Mapsforge project
   - Core offline map rendering stack in the watch app.
   - Project: https://github.com/mapsforge/mapsforge

12. MapLibre Native
   - Native online map picker used by the companion app for area and routing-tile selection.
   - Project: https://github.com/maplibre/maplibre-native

13. BRouter (Arndt Brenschede and contributors)
   - Offline routing engine used by the watch app, with routing packs downloaded by the companion app.
   - Project: https://github.com/abrensch/brouter

14. Mapbox Maki icons
   - Build-time POI icon source.
   - Project: https://github.com/mapbox/maki

## AI-assisted development acknowledgment

This app was primarily developed with AI-assisted coding tools.

We recognize that generative AI systems are trained on large corpora of human-created work, and generated outputs do not include direct attribution to every original contributor.

Project position:

- We support creator consent, proper licensing, attribution, and fair compensation practices.
- We prioritize open and clearly licensed data/sources for app features and map workflows.
- We provide project-level attribution and source links in this licenses section.
